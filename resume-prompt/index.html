자기소개서 구조설계 AI 프롬프트 생성기 (STAR)

희망기업·희망직무 기반 직무/기업 요구사항 요약 → (즉시) 예시 생성 → 부족정보 질문 → 보완 후 개선 순서로 프롬프트를 만듭니다.

✅ GitHub Pages 단일 파일 ✅ 요구사항 먼저 요약 ✅ 즉시 예시 먼저 생성 ✅ 부족정보만 추가 질문

입력

Ctrl + Enter 생성
Tip: JD 전문을 그대로 넣으면 ‘요구역량/키워드/성과지표’ 추출이 더 정확해집니다. (없어도 ‘희망기업/희망직무’ 기반 추론 + (가능하면) 리서치 지시를 포함합니다)
생성된 프롬프트를 그대로 생성형 AI에 붙여넣어 사용하세요.
주의: 본 도구는 ‘구조 설계/참고’ 목적입니다. 사실/수치/경험은 본인 기준으로 검증·보완해야 합니다.

출력 (AI에 그대로 복붙)

- 이 버전은 “질문 먼저”가 아니라 예시를 먼저 만든 뒤, 부족한 정보만 추가로 묻도록 설계했습니다.
- “미니프롬프트”는 빠른 테스트/반복에 유용합니다.
AI를 활용한 작성은 구조를 잡는 단순 참고과정일 뿐임으로, 기업 및 직무분석을 활용해 맞춤형 자소서를 완성하시기 바랍니다.
