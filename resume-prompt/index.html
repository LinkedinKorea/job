자기소개서 구조설계 AI 프롬프트 생성기 (STAR)

기본정보 + 희망직무/기업/업종 입력 → STAR 구조 설계 + 기업 맞춤 문장 생성 / (선택) 기존 자소서 코칭·수정 프롬프트를 자동 생성합니다.

✅ GitHub Pages 단일 파일 ✅ STAR 자동 설계 ✅ 기업 맞춤 톤/키워드 ✅ 선택: 기존 자소서 코칭

입력

자동저장: ON
Tip: JD 전문을 그대로 넣으면 맞춤 키워드 추출 정확도가 올라갑니다.
생성된 프롬프트를 그대로 ChatGPT/LLM에 붙여넣으면 됩니다.
권장: 모델에게 “추가 질문부터” 하게 해서 소재 부족을 먼저 보완하도록 설계했습니다.

출력 (AI에 그대로 복붙)

- “미니프롬프트”는 길이를 줄여 빠르게 돌릴 때 유용합니다.
- 본문이 길면, 생성 후 “1) 질문만 먼저 2) 최종본 작성” 순서로 진행하세요.
