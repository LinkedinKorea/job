자기소개서 구조설계 AI 프롬프트 생성기 (STAR)

희망기업·희망직무 기반 기업/직무 분석 → 항목 추천/구조 설계 → STAR 설계 → 짧은 예시까지 한 번에 프롬프트로 만듭니다.

✅ GitHub Pages 단일 파일 ✅ 기업/직무 분석 포함 ✅ 항목별 추천/STAR 구조 ✅ 짧은 예시 출력

입력

Ctrl + Enter 생성
Tip: JD 전문을 그대로 넣으면 맞춤 키워드 추출 정확도가 올라갑니다. (없어도 ‘희망기업/희망직무’ 기반 리서치 지시를 포함합니다)
생성된 프롬프트를 그대로 생성형 AI에 붙여넣어 사용하세요.
주의: 본 도구는 ‘구조 설계/참고’ 목적입니다. 사실/수치/경험은 본인 기준으로 검증·보완해야 합니다.

출력 (AI에 그대로 복붙)

- 길게 나오면: ① 질문만 먼저 받기 → ② 답변 후 최종본 작성 순서로 진행하세요.
- “미니프롬프트”는 빠른 테스트/반복에 유용합니다.
AI를 활용한 작성은 구조를 잡는 단순 참고과정일 뿐임으로, 기업 및 직무분석을 활용해 맞춤형 자소서를 완성하시기 바랍니다.
